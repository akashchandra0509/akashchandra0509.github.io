Virtual Mouse System

A personal project to create a hands-free method of computer control. This system uses hand gestures captured through a webcam to perform standard mouse operations, offering an intuitive and seamless user interaction experience.

  • Developed using Python, with the OpenCV and MediaPipe libraries for advanced computer vision.
  • Implemented real-time hand tracking to simulate cursor movement, clicks, scrolling, and drag-and-drop actions.
  • Utilized the PyAutoGUI library to map gesture recognition data to system-level mouse events.
